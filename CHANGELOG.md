## [0.0.1] - 2021-09-XX

### Adicionado
- Projeto inicial desenvolvido durante a Semana Spring React - SDS 3.0
- Backend com Spring Boot
  - API RESTful para vendas
  - Endpoints para listagem de vendas (paginado)
  - Endpoints para agregação de dados
  - Integração com H2 Database e PostgreSQL
- Frontend com React e TypeScript
  - Dashboard com gráficos interativos
  - Gráfico de barras para volume de vendas
  - Gráfico de rosca para taxa de sucesso
  - Tabela paginada de vendas
- Deploy
  - Backend no Heroku
  - Frontend no Netlify
